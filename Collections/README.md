Java Collections examples

Files in this folder demonstrate common collection implementations and their typical methods. To compile and run an example:

javac ArrayListExamples.java
java ArrayListExamples

Repeat for other files (replace filename accordingly).

Files:

- ArrayListExamples.java
- LinkedListExamples.java
- VectorExamples.java
- StackExamples.java
- HashSetExamples.java
- LinkedHashSetExamples.java
- TreeSetExamples.java
- HashMapExamples.java
- LinkedHashMapExamples.java
- TreeMapExamples.java
- PriorityQueueExamples.java
- DequeExamples.java

Each file contains short comments describing common methods and usage patterns.
